## Executive Summary
This report summarizes 19,620 events collected from the honeypot network. The majority of attacks targeted SSH (Cowrie), SIP (Sentrypeer), and general network services (Honeytrap). A significant number of alerts were generated by the Suricata IDS. Attackers were observed attempting to gain unauthorized access, download and execute malicious files, and add SSH keys for persistence.

## Key Observations and Anomalies
- The high number of Cowrie events indicates a focus on compromising SSH servers.
- A large number of SIP/VoIP attacks were observed, primarily targeting port 5060, and triggering the "ET VOIP MultiTech SIP UDP Overflow" signature (related to CVE-2005-4050).
- Attackers are consistently attempting to download and execute malicious files, including scripts named `w.sh`, `c.sh`, `wget.sh` and binaries like `arm.urbotnetisass` and `Mozi.m`.
- A common tactic observed is the attempt to add a specific SSH public key to the `authorized_keys` file for persistent access.
- Numerous commands were executed to gather system information (CPU, memory, OS) likely for fingerprinting the environment.
- There are several `nohup` commands attempting to establish reverse shells to IP `47.120.55.164`.
